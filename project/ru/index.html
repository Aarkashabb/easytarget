Частые вопросы

Распространённые вопросы о сотрудничестве и процессе автоматизации.

С какими платформами вы работаете?
В основном с N8N (self-hosted и облачным), Make.com, Zapier и прямыми API/SDK-интеграциями на Node.js и Python. Для AI-функций использую API OpenAI, Anthropic и Google AI.
Сколько времени занимает типичный проект?
Discovery обычно занимает неделю. Типичная автоматизация запускается за 2–6 недель — в зависимости от объёма и количества интегрируемых систем.
Вы оказываете поддержку после релиза?
Да — каждый запуск включает 30 дней постзапускной поддержки. Также предлагаю ежемесячные ретейнеры: мониторинг, оптимизация и добавление новых процессов.
Как вы обеспечиваете безопасность данных?
Все учётные данные хранятся в зашифрованных vault-хранилищах, интеграции используют ограниченные API-токены, а self-hosted развёртывание даёт полный контроль над расположением данных. NDA и DPA — по запросу.
Можете ли вы интегрироваться с legacy-системами?
Да. Если у системы есть API — интеграция возможна. Для более старых систем без API создаю кастомный middleware (обычно через N8N или тонкий слой на Node.js).
Как устроено ценообразование?
Работаю преимущественно по фиксированной цене проекта после Discovery — полная стоимость известна заранее. Ретейнеры — ежемесячная оплата. Почасовая — для небольших правок.
